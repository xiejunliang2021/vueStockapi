# Deploy Frontend Skill

这个 skill 包含了完整的 Vue3 前端项目部署流程和工具。

## 文件说明

- **SKILL.md**: 主要的 skill 文档，包含详细的使用说明和步骤
- **deploy-template.sh**: 部署脚本模板，可复制到项目中使用
- **quick-reference.md**: 快速参考指南，常用命令和配置
- **examples.md**: 使用示例和场景对话

## 快速开始

当用户请求部署前端时：

1. 阅读 `SKILL.md` 了解完整流程
2. 参考 `examples.md` 中的场景选择合适的步骤
3. 使用 `deploy-template.sh` 创建部署脚本
4. 参考 `quick-reference.md` 查找常用命令
